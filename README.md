# 枢子AgentBot 公开发布仓库

这个仓库只用于发布枢子AgentBot 的安装包、更新清单和校验文件。

## 下载

请从 Releases 页面下载最新版：

https://github.com/id100w/shuziai-agent/releases

Windows 用户优先下载：

- `ShuziAgentBot-vX.Y.Z-Windows-Setup.exe`：安装版
- `ShuziAgentBot-vX.Y.Z-Windows-Portable.zip`：绿色版，如该版本提供
- `SHA256SUMS.txt`：文件校验和

## 更新源

客户端会从本公开仓库检查更新：

- Release API: `https://api.github.com/repos/id100w/shuziai-agent/releases/latest`
- Tauri updater manifest: `https://github.com/id100w/shuziai-agent/releases/latest/download/latest.json`

## 源码说明

本仓库不包含源码。

枢子AgentBot 的源码仓库保持私有；本公开仓库只承接面向用户的下载、更新和发布说明。
